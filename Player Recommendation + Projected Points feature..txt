5. EXPECTED MINUTES MODEL
========================================

Expected minutes are very important.

Implement ExpectedMinutesCalculator.

Inputs:
- Average minutes over recent matches
- Starts in recent matches
- Injury/availability status
- Chance of playing from FPL data
- Suspension/red flag
- Rotation pattern
- Fixture congestion

Simple MVP logic:

If player status is unavailable:
ExpectedMinutes = 0

If chance_of_playing_next_round exists:
AvailabilityFactor = chance_of_playing_next_round / 100

RecentAverageMinutes = average minutes from available recent data

StartProbability =
recentStarts / recentMatches

ExpectedMinutes =
RecentAverageMinutes × AvailabilityFactor

Clamp ExpectedMinutes between 0 and 90.

Risk examples:
- 90 mins often => low risk
- 60-70 mins often => medium risk
- inconsistent starts => high risk
- injury flag => high risk
- suspended/unavailable => avoid

Return:
- ExpectedMinutes
- StartProbability
- MinutesRiskLevel: Low / Medium / High
- Explanation

6. FIXTURE DIFFICULTY MODEL
========================================

Implement FixtureDifficultyCalculator.

For each upcoming fixture:
Calculate a fixture multiplier based on:

- Home/away
- Opponent strength
- Player position
- Team attacking strength
- Team defensive strength
- FPL fixture difficulty if available
- Double gameweek
- Blank gameweek

Simple default multipliers:

Very easy fixture: 1.20
Easy fixture: 1.10
Medium fixture: 1.00
Hard fixture: 0.90
Very hard fixture: 0.80

Position-specific logic:
- Attackers benefit from weak opponent defence
- Defenders/GKs benefit from weak opponent attack
- Midfielders use mixed logic

Double Gameweek:
- Project both fixtures and add them together

Blank Gameweek:
- Project 0 points

Return:
- FixtureScore
- FixtureMultiplier
- FixtureDifficultyLabel
- FixtureExplanation

8. CLEAN SHEET / DEFENSIVE POINTS
========================================

Implement expected clean sheet points.

For GK/DEF:
- High clean sheet chance gives stronger projection
- Goals conceded risk reduces projection

For MID:
- Clean sheet point is smaller but still included

For FWD:
- No clean sheet points

Simple model:

CleanSheetExpectedPoints =
PositionCleanSheetPoints × CleanSheetProbability

PositionCleanSheetPoints:
- GK = 4
- DEF = 4
- MID = 1
- FWD = 0

CleanSheetProbability can be approximated from:
- Team defensive strength
- Opponent attacking strength
- Home/away
- Recent clean sheets
